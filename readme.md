# 🎵 Get Yandex Music Token – Локальный сервер для прослушивания музыки

---

## 📥 1. Скачивание архива

Перейдите в [релизный архив](https://github.com/Hazzz895/Get-Yandex-Music-Token/releases) и скачайте последнюю версию.

---

## ⚙️ 2. Настройка окружения (файл `.env`)

В корневой папке проекта создайте файл `.env` и заполните его своими данными:

```ini
SUPABASE_PRIVATE_KEY=секретный_ключ_от_supabase
SUPABASE_PUBLISH_KEY=публичный_ключ_от_supabase
SUPABASE_URL=https://iklenaxzcxsabsndsxcp.supabase.co
SECRET_KEY=ваш_секретный_ключ_для_JWT   # можно использовать хэш строки или случайный набор букв
ALGORITHM=HS256
YANDEX_TOKEN=токен_полученный_на_следующем_шаге
```

🎫 Получение YANDEX_TOKEN
Откройте скачанный архив.

Запустите файл GetYandexMusicToken.exe (или GetYandexMusicToken для других ОС).

Введите свои учётные данные от Яндекс.

Скопируйте полученную строку и вставьте её в поле YANDEX_TOKEN в файле .env.

🐍 3. Установка зависимостей и виртуальное окружение
Откройте cmd и перейдите в папку проекта через cd

Создайте виртуальное окружение:

bash
```python -m venv venv```


Активируйте окружение:```venv\Scripts\activate```

Установите зависимости:```pip install -r requirements.txt```

🚀 4. Запуск сервера ```python Server/main.py```

🔊 5. Запуск клиента (плеера) ```python Player/main.py```

🎧 6. Наслаждайтесь музыкой!
